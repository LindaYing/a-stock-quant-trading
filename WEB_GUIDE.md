# A股量化交易系统 - Web界面使用指南

## 🚀 快速启动

### 方法1：使用启动脚本（推荐）

```bash
cd ~/.openclaw/workspace/quant_system
./start.sh
```

### 方法2：直接运行

```bash
cd ~/.openclaw/workspace/quant_system
streamlit run app.py
```

## 📱 访问方式

### 电脑访问
打开浏览器，访问：`http://localhost:8501`

### 手机访问
1. 确保手机和电脑在同一 WiFi 网络
2. 启动后会显示网络地址，例如：`http://192.168.1.100:8501`
3. 手机浏览器输入该地址

## 🎨 界面功能

### 📊 侧边栏 - 策略配置

**数据设置**
- 选择数据源（模拟数据/真实数据）
- 股票代码输入
- 数据天数选择

**策略参数**
- 短期均线周期（默认 5 天）
- 长期均线周期（默认 20 天）
- 初始资金（默认 10 万）
- 手续费率（默认 0.03%）

**模拟数据参数**（仅模拟模式）
- 波动率调整
- 趋势调整

### 📈 主界面 - 结果展示

**关键指标卡片**
- 💰 策略收益率
- 📊 买入持有收益
- 📉 最大回撤
- 🔄 交易次数
- 🎯 胜率

**可视化图表**
1. **K线图与交易信号**
   - 蜡烛图
   - 短期/长期均线
   - 买入/卖出标记
   - 成交量柱状图

2. **资金曲线**
   - 策略资金走势
   - 买入持有基准对比

3. **回撤曲线**
   - 实时回撤监控
   - 最大回撤标注

**交易记录表**
- 完整交易历史
- 日期、动作、价格、股数
- 可下载 CSV

## 🎯 使用流程

### 1. 快速体验（模拟数据）

```
1. 启动应用
2. 侧边栏选择"模拟数据"
3. 调整策略参数（可选）
4. 点击"开始回测"
5. 查看结果
```

### 2. 真实数据回测

```
1. 确保已安装 akshare: pip3 install akshare
2. 侧边栏选择"真实数据"
3. 输入股票代码（如：000001）
4. 设置数据天数
5. 调整策略参数
6. 点击"开始回测"
7. 分析结果
```

## 🔧 参数调优建议

### 短期均线（MA Short）
- **5天**：灵敏，适合短线
- **10天**：平衡
- **20天**：稳健，适合中线

### 长期均线（MA Long）
- **20天**：短期趋势
- **60天**：中期趋势
- **120天**：长期趋势

### 常用组合
- 激进：MA(5, 20)
- 平衡：MA(10, 30)
- 稳健：MA(20, 60)
- 长线：MA(30, 120)

## 📊 结果解读

### 收益指标

**策略收益 > 买入持有**
✅ 策略有效，跑赢大盘

**策略收益 < 买入持有**
⚠️ 策略需优化，或市场不适合

### 风险指标

**最大回撤**
- < 10%：风险低
- 10-20%：风险中等
- > 20%：风险较高

**胜率**
- > 50%：策略较好
- 40-50%：可接受
- < 40%：需要优化

### 交易频率

**交易次数过多**
- 可能频繁交易，手续费高
- 考虑增大均线周期

**交易次数过少**
- 可能错过机会
- 考虑减小均线周期

## 🐛 常见问题

### Q: 启动后浏览器没有自动打开？
A: 手动访问 `http://localhost:8501`

### Q: 手机无法访问？
A: 
1. 确保同一 WiFi
2. 检查防火墙设置
3. 使用启动时显示的网络地址

### Q: 真实数据获取失败？
A: 
1. 检查网络连接
2. 确认股票代码正确
3. 尝试减少数据天数

### Q: 回测速度慢？
A: 
1. 减少数据天数
2. 使用模拟数据测试参数
3. 参数确定后再用真实数据

### Q: 图表显示不完整？
A: 
1. 刷新页面
2. 调整浏览器窗口大小
3. 尝试其他浏览器
